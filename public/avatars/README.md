# 头像库文件夹

这个文件夹用于存储预设的头像图片。

## 文件命名规范

- `avatar_001.png` - 温暖阳光
- `avatar_002.png` - 宁静月光  
- `avatar_003.png` - 活力火焰
- `avatar_004.png` - 智慧星辰
- `avatar_005.png` - 自然清风
- `avatar_006.png` - 梦幻彩虹

## 图片要求

- 格式：PNG（推荐）或 JPG
- 尺寸：建议 200x200 像素或更高
- 文件大小：建议小于 100KB
- 背景：透明或纯色背景

## 使用方法

1. 将头像图片放入此文件夹
2. 确保文件名与 `lib/avatar-library.ts` 中的配置一致
3. 图片将自动在个人资料页面中显示

## 注意事项

- 请确保图片版权合法
- 建议使用统一的设计风格
- 图片名称不要包含空格或特殊字符
